== Frequently Asked Questions ==

= What is keyword cannibalization? =  
It occurs when multiple posts on your website compete for the same keyword, confusing search engines and reducing your SEO potential.

= Does this plugin support custom post types? =  
Yes, it supports all public post types including posts, pages, and custom types.

= Can I connect this to ChatGPT? =  
Yes, the plugin allows OpenAI integration using your own API key for content analysis and recommendations.

= Does it modify my content automatically? =  
No. All changes require manual approval unless you choose bulk action mode.
